# OneFeed


### RAG + LLM:

#### Install requirements

`pip install -r llm_requirements.txt`

#### Set Mistral, MongoDB keys, and X user and password

Set the API keys in `.env`
```
ATLAS_URI=""
MISTRAL_API_KEY=""
USERNAME=""
PASSWORD=""
```

#### Query

Pass the query to `main.py` and execute <br />

python3 main.py()
